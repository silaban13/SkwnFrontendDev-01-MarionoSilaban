# SkwnFrontendDev-01-MarionoSilaban

Frontend project berbasis **Vue.js** untuk latihan pengembangan web interaktif dengan layout responsif, carousel produk, form interaktif, dan animasi scroll.  

## Deskripsi
Proyek ini menggunakan Vue.js sebagai framework frontend dengan **Vite** sebagai build tool. Data produk menggunakan **API mock** atau data random untuk menampilkan konten dinamis.  

Fitur utama:
- Layout responsif (mobile → desktop)  
- Carousel produk dengan tombol navigasi  
- Overlay teks dan tombol pada gambar produk  
- Form input interaktif (email / search)  
- Animasi AOS (Animate on Scroll)  
- Fetch data dari API mock atau generate random
